Blender

**Pop**

```
Shift+a - add
xd - delete

g - move
s - scale
r + [x/y/z]- rotate

tab - edit/object mode
1, 2, 3 - vertex, edge, face
f - create face

e - extrude
ctrl + right mouse click - extrude to mouse 

ctrl + r - loop cut (possible with scroll wheel)
k - knife tool

ctrl + b - bevel (rounding edges and vertex)
ctrl + shift + b - vertex bevel
```

**Options**

```
shift + d - duplicate

t - toolbar window
n - properties window
```

```
shift + c - reset cursor
alt + z - xray toogle

m - move to collection
```

```
ctrl + j - join object into one mesh
p - separate to mesh
```
